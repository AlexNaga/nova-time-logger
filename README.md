## Time logger

### How to run?

Copy the file `.env.example` , name it `.env` and change the values.

``` sh
$ npm i
$ npm run build
$ npm run client
```

### How to install globally?

By linking it globally, it's possible to run command `nova` in the terminal.

``` sh
# npm link
```

### How to remove globally?

``` sh
# npm unlink
```
